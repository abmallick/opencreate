# Test Datasets for Evals

## Required Test Images

Add the following images to run the evaluation suite:

### Subject Images (`subjects/`)

Add 3 product images with transparent or white backgrounds:

1. **subject-bottle.png** - A glass bottle (water/beverage)
2. **subject-mug.png** - A coffee mug or cup  
3. **subject-sneaker.png** - A sneaker or shoe

**Requirements:**
- PNG format
- Clear product isolation (white/transparent background preferred)
- High quality, well-lit product photos
- Minimum 512x512 pixels

### Scene Images (`scenes/`)

Add 4 background/scene images:

1. **scene-kitchen.png** - Kitchen counter or surface
2. **scene-outdoor.png** - Outdoor table with garden/nature
3. **scene-studio.png** - Minimalist studio setting
4. **scene-white.png** - Plain white or subtle gradient background

**Requirements:**
- PNG or JPG format
- Empty scenes without products (product will be composited in)
- Good for product placement
- Minimum 1024x1024 pixels

## Quick Start with Sample Images

If you don't have product images, you can use placeholder images for testing:

```bash
# Download sample placeholder images (you'll want to replace with real products)
cd evals/datasets

# Create placeholder files
mkdir -p subjects scenes

# Option 1: Use solid colors as placeholders
convert -size 512x512 xc:lightgray subjects/subject-bottle.png
convert -size 512x512 xc:lightgray subjects/subject-mug.png  
convert -size 512x512 xc:lightgray subjects/subject-sneaker.png
convert -size 1024x1024 xc:beige scenes/scene-kitchen.png
convert -size 1024x1024 xc:lightgreen scenes/scene-outdoor.png
convert -size 1024x1024 xc:white scenes/scene-studio.png
convert -size 1024x1024 xc:white scenes/scene-white.png
```
